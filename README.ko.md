# Lipbalm Reminder
## 구상
```
건조한 겨울
부르튼 입술
립밤을 바를
알림이 필요
```
## 구현
260204 - 구상, 프로젝트 생성

## 계획
- 버튼 - 알림 허용
- 설치 - PWA로 동작
- 서버액션 - 푸시 전송
- cron job - 주기적으로 서버액션 실행
- 추가 구현: 로그인, 개인화
